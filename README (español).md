# Vue.js Country Information App

Aplicación web desarrollada en Vue.js que permite a los usuarios explorar información detallada de todos los países del mundo utilizando una API pública. Este proyecto demuestra mi capacidad para implementar un diseño preexistente y crear un diseño responsive en Vue.js y Bootstrap partiendo de un enfoque desktop-first.


## Características principales

- **Listado de países**: Muestra una lista de países con sus banderas y nombres.
- **Detalles del país**: Al seleccionar un país, se muestra información detallada como capital, población, región, idiomas y más.
- **Renderización condicional**: Uso de v-if/v-else para alternar entre la lista de países y los detalles de un país seleccionado.
- **Comunicación entre componentes**: Implementación de props y emits para la comunicación entre componentes.
- **Diseño responsive personalizado**: Desarrollado con Bootstrap y mejorado con clases CSS personalizadas para ampliar su funcionalidad.


# Aspectos destacados

- **Implementación del diseño**: El diseño del proyecto fue proporcionado externamente, y lo traduje a código funcional, logrando una implementación fiel al diseño original.
- **Clases CSS personalizadas**: Aunque utilicé Bootstrap como base, creé clases CSS personalizadas para extender sus capacidades y lograr el diseño deseado.
- **Enfoque desktop-first**: El proceso de diseño comenzó con un layout para desktop, que luego se adaptó para ser completamente responsive en todos los dispositivos.


## Tecnologías utilizadas

- **Vue.js 3**: Para construir la interfaz de usuario.
- **Bootstrap**: Para el diseño y la estructura responsive, complementado con CSS personalizado.
- **Rest Countries API**: Para obtener los datos de los países almacenados en directorio local.
- **HTML/CSS**: Para estilos adicionales y personalización.

## Cómo funciona

1. **Vista inicial**: Muestra una lista de países.
2. **Selección de país**: Al hacer clic en un país, se muestran sus detalles.
3. **Comunicación entre componentes**: Los datos del país seleccionado se pasan como prop al componente de detalles, que luego muestra la información.


# ¿Por qué este proyecto?
